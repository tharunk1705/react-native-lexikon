# React Native - Lexikon

- This is a simple dictionary app created with react native.

- This project uses the FreeDictionaryAPI [https://dictionaryapi.dev/]

## Run Andriod

```bash
  npx react-native run-Andriod
```

## Run IOS

```bash
  npx react-native run-ios
```

## Run test

```bash
  npm run test
```
